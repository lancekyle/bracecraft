REPLIT STARTER PROMPT: BraceCraft Authority Site

Create a modern, mobile-first website for BraceCraft - an athletic injury prevention and recovery authority site.

Tech Stack Requirements

•
Framework: React (latest version)

•
Styling: Tailwind CSS

•
Routing: React Router for multi-page navigation

•
Deployment: Replit hosting

•
Performance: Optimize for fast loading and mobile-first design
